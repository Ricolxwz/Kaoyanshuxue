# Kaoyanshuxue
此仓库为考研数学二`LaTeX`仓库, 提供线性代数和高等数学`LaTeX`代码, 可根据自己情况编译.

如果不想自己编译的话, 可以直接去旁边的`Release`下载`main.pdf`文件.

另外, 本项目代码已超过3000行, 可以的话点个🌟支持一下让更多人发现~
